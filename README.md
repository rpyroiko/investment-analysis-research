# Investment Analysis Research

## Overview
This project focuses on the comparative analysis of two major tech companies, Tesla (TSLA) and Apple (AAPL), to understand their investment potential.

## Features
- Comparative financial metrics
- Visualizations of stock performance
- Analysis of market trends

## Tech Stack
- Python
- Pandas
- Matplotlib

## Usage
Clone the repository and run the analysis script:
```bash
python analysis.py
```
